Array.prototype.filter()
        filter() 메서드는 조건에 맞는거에 대한 새로운 배열로 반환합니다.

        구문
        arr.filter(callback(element[, index[, array]])[, thisArg])

        arr.filter(function(현재값){
            return 조건
        })

        arr.filter((현재값) => {
            return 조건
        })

        arr.filter((현재값) => return 조건)

        callback : 함수. 
        true를 반환하면 요소를 유지하고, false를 반환하면 버립니다.

        다음 세 가지 매개변수를 받습니다.

        element : 현재 요소.

        반환 값
        테스트를 통과(조건)한 요소로 이루어진 새로운 배열.
